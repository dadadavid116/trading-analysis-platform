# Trading Analysis Platform

## Overview
A VPS-hosted crypto market monitoring platform with multiple visual panels for price, liquidation, liquidity, alerts, and AI-assisted analysis.

## Goal
Build a personal market intelligence dashboard first, then evolve it into a shareable and customizable platform.

## MVP Features
- Live market monitoring
- Multiple dashboard panels
- Liquidation visualization
- Liquidity / order-book monitoring
- Historical data storage
- AI-assisted market summaries
- Alerts

## Planned Stack
- VPS hosting
- Backend collectors and API
- Database
- Frontend dashboard
- Claude-assisted development
- Later: multi-model support for logic and news
